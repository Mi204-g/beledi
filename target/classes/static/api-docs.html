Signalements

GET /api/signalements

Récupérer tous les signalements (public)

GET /api/signalements/page

Récupérer les signalements avec pagination

Paramètres:

  • page — Numéro de page (défaut: 0)
  • size — Éléments par page (défaut: 10)
  • sortBy — Champ de tri (défaut: dateCreation)
  • descending — Tri descendant (défaut: true)
GET /api/signalements/{id}

Récupérer un signalement spécifique (public)

GET /api/signalements/mes

Récupérer mes signalements (authentifié)

Nécessite: Authorization: Bearer {token}

POST /api/signalements

Créer un nouveau signalement (authentifié)

{
  "titre": "Nid de poule dangereux",
  "description": "Grand trou sur la route principale",
  "categorie": "VOIRIE",
  "latitude": 18.0785,
  "longitude": -15.9654,
  "photoUrl": "https://xxx.com/photo.jpg"
}
PUT /api/signalements/{id}

Mettre à jour un signalement (authentifié - auteur ou admin)

Même format que POST

PUT /api/signalements/{id}/statut

Mettre à jour le statut (admin seulement)

{
  "statut": "EN_COURS"
}

Valeurs: EN_ATTENTE, EN_COURS, RESOLU

DELETE /api/signalements/{id}

Supprimer un signalement (authentifié - auteur ou admin)

Commentaires

GET /api/signalements/{id}/commentaires

Récupérer tous les commentaires d'un signalement (public)

GET /api/signalements/{id}/commentaires/{commentaireId}

Récupérer un commentaire spécifique (public)

POST /api/signalements/{id}/commentaires

Ajouter un commentaire (authentifié)

{
  "contenu": "Merci d'avoir signalé ce problème!"
}
PUT /api/signalements/{id}/commentaires/{commentaireId}

Mettre à jour un commentaire (authentifié - auteur ou admin)

DELETE /api/signalements/{id}/commentaires/{commentaireId}

Supprimer un commentaire (authentifié - auteur ou admin)

Dashboard Admin

GET /api/admin/statistiques

Récupérer les statistiques du dashboard (admin seulement)

Retourne: counts par statut, counts par catégorie, total

Valeurs Énumérées

Catégories

  • VOIRIE — Routes et voirie
  • ELECTRICITE — Électricité
  • DECHETS — Déchets
  • EAU — Adduction d'eau
  • AUTRE — Autre

Statuts

  • EN_ATTENTE — En attente
  • EN_COURS — En cours de traitement
  • RESOLU — Résolu
